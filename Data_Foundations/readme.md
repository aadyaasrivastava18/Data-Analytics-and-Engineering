# 📚 Data Foundations

This section covers the core concepts required before learning SQL, Data Modeling, Data Engineering, and Analytics.

## Topics Covered

- Data Pipeline
- Data Lake
- Data Warehouse
- OLTP vs OLAP
- SSOT (Single Source of Truth)
- ACID Properties

## Learning Outcomes

After completing this section, you should be able to:

- Explain the data lifecycle.
- Differentiate between Data Lakes and Data Warehouses.
- Understand why Data Pipelines exist.
- Explain why organizations need centralized business logic.
- Distinguish between OLTP and OLAP systems.
- Understand the importance of SSOT in large organizations.
